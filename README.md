# Quickstart
Fastest way to run the app.

1. Install Expo
```
npm install -g expo-cli
```

2. Download Repo
```
git clone https://github.com/Foodrop/mobile.git
cd mobile
```

3. Install everything
```
npm install
```

4. Starting the app
```
expo start
```

5. The rest is history.

If you hit an error, please submit an Github Issue.
